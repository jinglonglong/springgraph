## Middleware Types Supported
- Database: spring.datasource.* (mysql, postgresql, oracle, sqlserver)
- Cache: spring.redis.*
- MQ: spring.kafka.*, spring.rabbitmq.*
- Search: spring.elasticsearch.*
- Job Scheduler: xxl.job.*
- Object Storage: minio.*, oss.*

## Key Features
- Infers middleware from runtime_config_properties (populated by T15)
- Creates CONNECTS_TO edges from micro_service to middleware
- Supports multi-datasource patterns (spring.datasource.order.url, spring.datasource.bill.url)
- Deterministic IDs using sha256 hash
