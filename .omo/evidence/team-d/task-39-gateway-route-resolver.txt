## Implementation Summary

## Key Features
- Parses spring.cloud.gateway.routes[] from YAML/properties
- Creates gateway_route symbols with routeId, uri, predicates, filters
- Creates ROUTES_TO edges to micro_service (lb://) or route_target (http(s)://)
- Creates MATCHES_PATH edges to endpoint symbols for Path predicates
- Supports lb://, http://, https://, ws://, wss:// URI schemes
- Detects gateway services by name pattern (*-gateway) or spring.cloud.gateway block
